# AI Chat API
> A FastAPI-based REST API for intelligent conversations with LLMs.

## Features
- Multi-turn conversations with memory
- Multiple chat sessions
- LLM-powered responses
- Clean REST API with auto-documentation

## Tech Stack
- FastAPI
- LangChain
- OpenAI/LLaMA
- SQLite

# React - explained
`main.jsx` - entry point aplikacije
`App.jsx` - glavna komponenta
`App.css` i `index.css` - still, index je obično globalni still, app je stil samo app komponente
